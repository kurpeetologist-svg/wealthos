# WealthOS Context Library v1.0

Behind the Number is powered by a reusable Context Library rather than isolated hardcoded messages.

## Purpose

The library maps a user action to:

- a concise financial concept
- a brief explanation
- optional aliases that help identify related merchants or descriptions

## Structure

### Expense contexts

Current concepts include:

- Coffee → Discretionary spending
- Food & Dining → Variable expenses
- Groceries → Essential but variable
- Transportation → Mixed fixed and variable expenses
- Housing → Fixed expenses
- Utilities → Recurring obligations
- Shopping → Discretionary or necessary
- Health → Predictable and unpredictable costs
- Entertainment → Discretionary spending
- Travel → Sinking funds
- Subscriptions → Recurring expenses
- Taxes → Tax payments
- Debt → Principal and interest
- Insurance → Risk transfer
- Investment → Risk and return

### Check-in contexts

- Weekly → Reference point versus pattern
- Monthly with saving → Saving within cash flow
- Monthly spending above income → Negative cash flow
- Standard monthly → Cash flow

### Future context groups

- savings contributions
- emergency funds
- income and paychecks
- debt payments
- tax filing
- benefits
- insurance claims
- investments
- milestones

## Product rules

1. Context appears because of an action.
2. Context should take only a few seconds to understand.
3. Context explains; it does not praise or judge.
4. Context is not another Lesson.
5. Context should not require another click to deliver its core value.
6. When uncertain, use a broader accurate concept rather than a specific unsupported claim.
7. Context should be expandable through data, not through redesigning the UI.
