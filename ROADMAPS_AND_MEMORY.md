# WealthOS Roadmaps & Financial Memory v1.0

## Financial Memory

Financial Memory is the long-term narrative layer of WealthOS.

It answers:

> What has changed in my financial life?

Initial Moment types:

- First purchase
- First Weekly Check-in
- First Monthly Check-in
- Three-week continuity
- Roadmap at 25%
- Roadmap at 50%
- Roadmap at 75%
- Roadmap completed

Moments are quiet records. They are not achievements, badges, points, or streak rewards.

## Roadmaps

Roadmaps translate life goals into financial direction.

They answer:

> Where am I going?

Each Roadmap stores:

- name
- type
- target amount
- amount completed or saved
- start date
- optional target date
- planned monthly contribution
- status

## Opportunity-cost foundation

Roadmaps create the future destination needed for later Purchase Context.

A future version may explain:

- how a repeated purchase changes a Roadmap timeline
- how a spending pattern affects a target
- how an additional contribution changes estimated completion

This must remain contextual and nonjudgmental.

## Migration

The former Savings Challenge becomes the first Roadmap automatically.
