# WealthOS v0.8.5 — User Testing Release

## Experience updates

- Default user name removed.
- The Lobby now greets everyone universally:
  - Good morning.
  - Good afternoon.
  - Good evening.
- Added rotating Morning Thoughts:
  - Let's see where you stand today.
  - Let's focus on what matters today.
  - One decision at a time.
  - Progress begins with clarity.

## Principle

**The Lobby welcomes. It doesn't analyze.**

The Lobby exists to help users arrive before WealthOS begins the financial conversation.

This version is intended for usability testing with real users.
